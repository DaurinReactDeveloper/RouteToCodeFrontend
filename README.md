<h1>🌐 Descubre Route to Code: La Senda del Código</h1>

<h2>🎨 Descripción del Proyecto (Frontend)</h2>
<p>El frontend de <strong>Route to Code</strong> está desarrollado con <strong>React.js</strong>, aprovechando la potencia de <strong>Vite</strong> para una construcción rápida y eficiente. Con una interfaz intuitiva y un diseño responsivo, la plataforma permite explorar diversas hojas de ruta de desarrollo de manera fluida y dinámica.</p>

<p>La aplicación se comunica con el backend mediante <strong>Axios</strong>, asegurando peticiones eficientes y una experiencia de usuario sin interrupciones. La autenticación se gestiona con <strong>jwt-decode</strong>, garantizando un acceso seguro y personalizado según el rol del usuario.</p>

<h2>🚀 Tecnologías Utilizadas</h2>
<ul>
    <li>⚛️ <strong>React.js</strong>: Construcción de interfaces modernas y dinámicas.</li>
    <li>⚡ <strong>Vite</strong>: Herramienta de construcción ultrarrápida para optimizar el rendimiento.</li>
    <li>🎨 <strong>Bootstrap</strong>: Estilos responsivos y componentes listos para usar.</li>
    <li>🔗 <strong>Axios</strong>: Manejo eficiente de peticiones HTTP al backend.</li>
    <li>🔐 <strong>jwt-decode</strong>: Decodificación y validación de tokens JWT para la autenticación segura.</li>
</ul>

<h2>🌍 Funcionalidades Destacadas</h2>
<ul>
    <li>📌 Navegación fluida y optimizada con React Router.</li>
    <li>📱 Diseño responsivo para una experiencia óptima en dispositivos móviles y de escritorio.</li>
    <li>🔑 Sistema de autenticación seguro basado en JWT.</li>
    <li>📄 Sección interactiva con hojas de ruta para guiar el aprendizaje en desarrollo.</li>
    <li>⚡ Carga rápida y optimización gracias a Vite.</li>
</ul>
